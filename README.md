# Exercícios de Lógica de Programação em C#

Este repositório contém uma lista de exercícios com o objetivo de exercitar o conteúdo visto no módulo de lógica de programação. Os exercícios foram implementados em C#.

## Exercício 1 - Conversor de Temperatura
Crie um programa em C# que permite ao usuário escolher entre converter uma temperatura de Celsius para Fahrenheit ou de Fahrenheit para Celsius. Solicite o valor a ser convertido e apresente o resultado.

## Exercício 2 - Verificador de Número Primo
Desenvolva um programa em C# que verifica se um número informado pelo usuário é primo ou não. Imprima uma mensagem indicando o resultado.

## Exercício 3 - Cálculo do Fatorial
Escreva um programa em C# que recebe um número inteiro positivo e calcula o seu fatorial. Imprima o resultado.

## Exercício 4 - Ordenação de Array
Desenvolva um programa em C# que ordena um array de números informado pelo usuário em ordem crescente. Imprima o array ordenado.

## Exercício 5 - Verificador de Palíndromo
Faça um programa em C# que verifica se uma palavra informada pelo usuário é um palíndromo. Um palíndromo é uma palavra que pode ser lida da mesma forma de trás para frente. Exemplo: "ovo", "arara".

## Exercício 6 - Cálculo da Raiz Quadrada
Implemente um programa em C# que calcula a raiz quadrada de um número informado pelo usuário sem utilizar funções específicas de bibliotecas.

## Exercício 7 - Conversor de Moeda
Desenvolva um programa em C# que converte um valor em reais para dólar. Solicite o valor em reais e a cotação do dólar e mostre o valor correspondente em dólares.

## Exercício 8 - Validador de Senha
Escreva um programa em C# que pede ao usuário para criar uma senha. Valide se a senha atende aos seguintes critérios: ter no mínimo 8 caracteres, conter pelo menos uma letra maiúscula, uma letra minúscula e um número. Indique se a senha é válida ou não.

## Exercício 9 - Validador de CPF
Desenvolva um programa em C# que valida se um número de CPF é válido. O usuário deve inserir um CPF, e o programa deve verificar se ele está no formato correto e se passa no critério de validação do dígito verificador.

## Exercício 10 - Análise de Frequência de Palavras
Desenvolva um programa em C# que receba um texto do usuário e faça uma análise, contando quantas vezes cada palavra aparece no texto e exibindo a frequência das palavras.

Boa sorte! ??
